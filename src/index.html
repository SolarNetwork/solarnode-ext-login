SolarNode External Login Demo

This page demonstrates how to integrate SolarNode into an external application, so that authentication can be performed outside SolarNode such that the SolarNode credentials are not directly passed to SolarNode. The source code for this app is available on GitHub. Note: most browsers will block loading mixed HTTPS/HTTP content, which can prevent this demo from working when loaded on a HTTPS URL. To work around this, grab the source and build the application and load it either as a file or via the development web server, as shown in the project's README.
